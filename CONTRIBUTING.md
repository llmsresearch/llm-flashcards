# Contributing

Thanks for taking an interest. This repository is a sample set of flashcards, so contributions are about quality and coverage rather than code.

## Found a mistake?

If a card is wrong, misleading, or unclear, please [open an issue](../../issues/new). Include the card name and what you think is off. Accuracy matters more to us than almost anything else, and reader corrections have improved the deck more than once.

## Want a topic covered?

Open an issue describing the concept you would like to see drawn. We keep a running list of requested topics and draw the ones that earn their own card. There is no guarantee, but every suggestion is read.

## A note on the images

The cards are released under [CC BY-NC-ND 4.0](LICENSE). Because the license is NoDerivatives, we cannot accept pull requests that modify the card images themselves. Corrections happen on our side; your issue is what triggers them.
